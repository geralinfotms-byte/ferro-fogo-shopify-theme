# Ferro & Fogo - Tema Shopify

Tema profissional para loja de equipamentos de serralheria e soldagem.

## Características

- Design industrial moderno (laranja, preto e branco)
- Layout responsivo para todos os dispositivos
- Carrinho drawer (sidebar)
- Seções customizáveis via Theme Editor
- Tradução completa em português (pt-BR)
- Otimizado para SEO

## Estrutura

```
├── assets/          # CSS e JavaScript
├── config/          # Configurações do tema
├── layout/          # Layout principal (theme.liquid)
├── locales/         # Traduções
├── sections/        # Seções reutilizáveis
├── snippets/        # Componentes pequenos
└── templates/       # Templates de páginas
```

## Instalação via GitHub

1. Faça fork deste repositório para sua conta GitHub
2. No Shopify Admin, vá em **Loja Online > Temas**
3. Clique em **Adicionar tema > Conectar do GitHub**
4. Selecione o repositório e a branch `main`
5. Clique em **Conectar**

## Configuração

Após instalar, customize o tema em **Personalizar**:

- **Hero Banner**: Imagem de fundo, textos e botões
- **Grade de Categorias**: Adicione suas coleções
- **Produtos em Destaque**: Selecione uma coleção
- **Footer**: Informações de contato

## Coleções Recomendadas

- `serralheiros` - Acessórios para Serralheiros
- `soldadores` - Acessórios para Soldadores  
- `equipamentos` - Equipamentos Diversos
